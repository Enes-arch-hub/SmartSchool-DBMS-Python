# 🎓 Smart School DBMS (Python)



## 📌 Overview



This is a School Management System built using Python and SQLite.  
It demonstrates database connectivity, modular programming, and data structures.

---


## 🚀 Features
- Student Management (CRUD)
- Course Management
- Fee Tracking System
- Library System
- Performance Analytics (Top Students using Heap)
- Streamlit Web Dashboard
- Login Authentication

----


## 🛠️ Technologies
- Python
- SQLite
- Streamlit
- Pandas
